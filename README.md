# ポートフォリオサイト

自分用のポートフォリオサイトです。HTML/CSSで作成されており、モダンでレスポンシブなデザインを採用しています。

## 特徴

- 🎨 モダンで美しいデザイン
- 📱 完全レスポンシブ対応（モバイル、タブレット、デスクトップ）
- ⚡ スムーススクロール機能
- 🍔 ハンバーガーメニュー（モバイル対応）
- 📧 お問い合わせフォーム
- 🎯 セマンティックHTML構造

## セクション

1. **ヒーローセクション** - 自己紹介とCTAボタン
2. **自己紹介** - プロフィールと統計情報
3. **スキル** - 技術スタックの紹介
4. **プロジェクト** - 作品集
5. **お問い合わせ** - 連絡先とフォーム

## ファイル構成

```
my-portforio/
├── index.html      # メインのHTMLファイル
├── styles.css      # スタイルシート
└── README.md       # プロジェクト説明
```

## 使い方

1. `index.html`をブラウザで開く
2. 必要に応じて以下の内容をカスタマイズ：
   - 名前やプロフィール情報
   - スキルリスト
   - プロジェクト情報
   - 連絡先情報

## カスタマイズ方法

### 個人情報の変更

`index.html`内の以下の部分を編集してください：

- 名前: `<span class="highlight">Your Name</span>`
- 職業: `<p class="hero-subtitle">Web Developer & Designer</p>`
- 統計情報: `.stat-number`の数値
- スキル: `.skill-card`内のリスト
- プロジェクト: `.project-card`の内容
- 連絡先: `.contact-item`内の情報

### カラーテーマの変更

`styles.css`の`:root`セクションでカラーパレットを変更できます：

```css
:root {
    --primary-color: #6366f1;    /* メインカラー */
    --secondary-color: #8b5cf6;  /* サブカラー */
    --text-color: #1f2937;       /* テキストカラー */
    /* ... */
}
```
